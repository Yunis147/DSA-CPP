# DSA-CPP

This repository is set up to store **Data Structures and Algorithms (DSA)** solutions in **C++**.

## Suggested structure

- `solutions/` → your C++ solution files

You can organize files by topic, for example:

- `solutions/arrays/`
- `solutions/linked-list/`
- `solutions/graphs/`

## File naming idea

Use clear names such as:

- `two_sum.cpp`
- `binary_search.cpp`
- `dijkstra_shortest_path.cpp`

## Compile and run a file

```bash
g++ -std=c++17 -O2 -Wall -Wextra solutions/<topic>/<file>.cpp -o solution
./solution
```
